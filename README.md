# 项目背景
本文为Nature Communications2025 年发表的研究论文，聚焦全球变暖背景下极端气候事件发生规律的周期性变化，填补了极端事件 “周期性特征” 及 “变暖对周期规律影响” 的研究空白，为气候适应规划提供全新分析视角。  
一、研究核心问题  
1.粮食歉收、热浪、野火三类极端气候事件是否存在稳定主导周期？  
2.从工业革命前到人类世，全球变暖如何改变这些周期性规律？  
3.极端事件的周期变化对气候适应、灾害防控有何关键启示？
# 复现目标
1.量化工业革命前极端事件的主导周期分布，明确周期与厄尔尼诺 - 南方涛动（ENSO）等气候振荡的关联。  
2.揭示全球变暖驱动下，极端事件主导周期的数量、时长、空间格局的演变规律。  
3.验证 “线性去趋势” 方法对分离变暖趋势与自然周期信号的有效性，复现周期向更短尺度转移的核心结论。
